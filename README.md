# RupeeSpent

RupeeSpent is a simple personal finance tracker to record, organize, and review day-to-day transactions.

## Features

- **Ledger**: Add, edit, delete, and reorder transactions within a day.
- **Statistics**: Monthly spend, category breakdown, and top spends.
- **Import/Export**:
  - Import from files (CSV/XLS/PDF) and resolve duplicates/conflicts safely.
  - Export your data for backup or analysis.
- **Date format preference**: Choose a date format and apply it across the app.
- **Financial Year (FY) switching**: Quickly switch between FY files/sheets.
- **Optional Google Drive/Sheets sync**: Store your ledger in your own Google Sheet for cross-device access.
- **Share Extension**: Import receipt/transaction info via the iOS share sheet (where supported).

## Data storage

- **Local mode**: Transactions are stored on-device in a CSV file (per Financial Year).
- **Google mode (optional)**: Transactions are stored in a Google Sheet owned by you, inside a folder you select.
